## :two: Getting Started

- <a href="https://classic.yarnpkg.com/en/" target="_blank">yarn 공식 홈페이지</a>에서 yarn 설치 후 아래 명령어 입력하여 로컬 상에서 서버 실행

```bash
yarn add
yarn start
```
